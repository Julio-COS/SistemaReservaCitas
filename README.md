EJECUCION EN LOCAL
1 Tener instalado XAMPP
2 Iniciar los módulos Apache y MySQL desde el panel de control.
3 Ubicar el proyecto dentro de htdocs
4 Luego acceder mediante el navegador usando la ruta del nombre del proyecto puesto en del htdocs


DESPLEGACION NUBE
Crear dominio o subdominio 
Subir archivos en el dominio de public_html/
Crear una nueva base de datos desde MySQL Databases
Actualizar la configuración del archivo de conexión (CREDENCIALES)
Acceder mediante el dominio (url)
